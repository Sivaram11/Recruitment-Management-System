# Recruitment-Management-System
Worked on the database design for Recruitment Tracking System which would assist the recruitment officers of an organization to post job requirements, select candidates and schedule interviews and take decisions on the process followed. The workflow involves from identifying functionalities the application should perform, end users, formulating Entity-Relationship design, determining the conceptual level, external and internal views and documenting the data dictionary under the supervision Dr. Martin Maskarinec.

Implemented the tables in 3NF in Oracle 11g and developed a console based application using technologies like Java and JDBC which is responsible to perform the activities like 

	Performing the DML activities on the database
	Performing the DDL activities on the database
	Displaying Menu Options to end users which performs complex Joins, and advance SQL 
	features and return the desired result to the users.
	Data obsolescence
